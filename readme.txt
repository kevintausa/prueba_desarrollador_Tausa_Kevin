Agend aApp:
Una persona es la encargada de gestionar (Crear, Actualizar, Eliminar y Mostrar) la información de
un usuario (identificación, nombres, apellidos, fecha de nacimiento, género). Cada usuario tiene
unos contactos que también podrán ser administrados (Crear, Actualizar, Eliminar y Mostrar). Cada
uno de los contactos del usuario consta de la siguiente información: nombre, número de contacto,
tipo de número (móvil, fijo) y parentesco (amigo, familiar o compañero de trabajo).
Usted debe crear una aplicación web que permita visualizar toda la información correspondiente a
los usuarios y contactos de cada usuario.
